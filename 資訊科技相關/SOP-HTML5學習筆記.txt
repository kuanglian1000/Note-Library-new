  > Using CSS
    #  CSS can be added to HTML documents in 3 ways:

        Inline - by using the style attribute inside HTML elements
        Internal - by using a <style> element in the <head> section
        External - by using a <link> element to link to an external CSS file

        The most common way to add CSS, is to keep the styles in "external CSS files".
        註1: 最常用的方式, 是外部CSS檔案.

    # Tip: With an "external style sheet", you can change the look of an entire web site, 
      by changing one file!

      註2: 外部CSS檔案, 可以在一個CSS檔案內, 改變整個網站的外觀.

      註3: 外部CSS檔案, HTML 的註解在CSS檔案不適用 <!-- <style>   </style> --> 

      註4: 外部CSS檔案, CSS 檔內的註解, 換成 /*--*-/

      註5: 外部CSS檔案, CSS關鍵字也更容易挑選, 顏色還會顯示在旁邊.
            body {background-color: powderblue;}
            body { background-color: lightyellow; }
            h1 { color: blue;}
            p  { color: red;}

    # CSS Colors, Fonts and Sizes

        The CSS "color" property defines the text color to be used. <文字顏色>

        The CSS "font-size: 600%;" property defines the text size to be used. <文字大小>

        The CSS "font-family" property defines the font to be used. <字型>

    # CSS Border
        The CSS border property defines a border around an HTML element.<外框>

        註=>Tip: You can define a border for nearly all HTML elements.

        p{
          border: 2px solid blue;
        }

    # CSS Padding
        The CSS padding property defines a padding (space) between the text and the border.
          (定義文字與外框間空白距離)

          p{
            border: 2px solid blue;
            padding: 30px;
          }

    # CSS Margin
        The CSS margin property defines a margin (space) outside the border.
          (定義外框外與其他元素間的空白距離)

          p{
            border: 2px solid blue;
            margin: 50px;
          }

    # Link to External CSS
      External style sheets can be referenced with a full URL 
        or with a path relative to the current web page.

        註: 連結外部完整的URL連結 OR 網站內部的CSS檔案.

       <link rel="stylesheet" href="https://www.w3schools.com/html/styles.css"> 
       <link rel="stylesheet" href="styles.css"> 

> HTML Links
  + HTML Links - Hyperlinks
      . HTML Links 是超連結.
      . 點選 連結, 跳至另一個文件.
      . 移至 連結上, 遊標會變成小手.
      註: Note: A link does not have to be text. 
                A link can be an image or any other HTML element!

      > HTML Links 語法 href Attribute
        <a href="URL">link text</a>
        <a href="https://eng.taiwan.net.tw/">Welcome to taiwan.</a>

        By default, links will appear as follows in all browsers:(連結預設樣式)
        An unvisited link is underlined and blue (藍)
        A visited link is underlined and purple (紫)
        An active link is underlined and red (紅)

      > HTML Links - The target Attribute

        The target attribute specifies where to open the linked document.
          註: 連結網頁開啟的方式

          The target attribute can have one of the following values:

              _self - Default. Opens the document in the same window/tab as it was clicked
                預設: 連結網頁開在原視窗內(可能只是parent frame 的一小部份)(網址不變)
              _blank - Opens the document in a new window or tab
                連結網頁開在新視窗內.(網址不變)

              _parent - Opens the document in the parent frame
                連結網頁開在父視窗內.(網址變成連結網頁)
              _top - Opens the document in the full body of the window
                連結網頁開在父視窗內.(網址變成連結網頁)


      > Absolute URLs vs. Relative URLs
        Absolute URLs 絕對路徑(通常是外部網站的路徑)      
            <h2>Absolute URLs</h2>
            <p><a href="https://www.w3.org/">W3C</a></p>
            <p><a href="https://www.google.com/">Google</a></p>

            <!-- 同網站檔案也可以用 '絕對路徑' 表示-->
             <a href="https://www.w3schools.com/html/default.asp">HTML tutorial</a> 

        Relative URLs 相對路徑(通常是內部網站的路徑)              
          <h2>Relative URLs</h2>
          
          <!-- 同網站相同目錄下的檔案當然可以用 '相對路徑' 表示-->
          <p><a href="html_images.asp">HTML Images</a></p>

          <!-- 同網站不同目錄下的檔案當然可以用 '相對路徑' 表示-->
          <p><a href="/css/default.asp">CSS Tutorial</a></p> 
           <a href="/html/default.asp">HTML tutorial</a>

  + Width and Height, or Style?
      The width, height, and style attributes are all valid in HTML.
      
    註: 建議作法使用 元素內的style 屬性來表示標註外觀.
      However, we suggest using the "style attribute". 
      It prevents styles sheets from changing the size of images:

        <style>
        img {
          width: 100%;
        }
        
        <!--這個寫法會變形-->
        <img src="html5.gif" alt="HTML5 Icon" width="128" height="128">
        
        <!--建議作法: 圖片不會變形, prevents styles sheets from changing the size of images-->
        <img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

        實際範例
          https://www.w3schools.com/html/tryit.asp?filename=tryhtml_images_style

       <p><img src="smiley.gif" alt="Smiley face" style="float:right;width:42px;height:42px;">
      The image will float to the right of the text.</p>

      <p><img src="smiley.gif" alt="Smiley face" style="float:left;width:42px;height:42px;">
      The image will float to the left of the text.</p>

    # HTML Table Colgroup
        The <colgroup> element is used to style specific columns of a table.
        用來針對部分欄位進行外觀設定

      > HTML Table Colgroup
          If you want to style the two first columns of a table, use the <colgroup> and <col> elements. 
            * <colgroup> 是 container
            * <col> 要放在 <colgroup>內
            * The 'span' attribute specifies how many columns that get the style.
              指定哪幾欄套用相同外觀設定
            * The 'style' attribute specifies the style to give the columns.
              要用外觀設定值

      註: Note: There is a very limited selection of legal CSS properties for colgroups.
        針對 colgroups 可設定的CSS值不多, 以下是少許合法的選項
          <width> property
          <visibility> property
          <background> properties
          <border> properties

          <table> 
            <colgroup>
              <col span="2" style="background-color: #D6EEEE">
            </colgroup>
            <tr>
              <th>MON</th>
              <th>TUE</th>
              <th>WED</th>
              <th>THU</th>
            </tr>
          </table>

      註: Note: The <colgroup> tag must be a child of a <table> element 
          and should be placed before any other table elements, 
          like <thead>, <tr>, <td> etc., 
          but after the <caption> element, 
          if present.
      
      註: <colgroup>的位置必須放在<table>內, <caption> 後面, 
          所有 其他 table 元素前, <thead> <tr> <td>

      > Multiple Col Elements

          在 <colgroup> 內, 使用多個 <col> elements.

        <table> /*這個資料夾總共有 9列(2+3+4=9)*/

+ HTML Lists

    > Unordered HTML List <ul>
      An 'unordered' list starts with the <ul> tag. 
      Each list item starts with the <li> tag.
      The list items will be marked with bullets (small black circles) by default:
        預設項目以'小圓黑點'標註
      <ul>
        <li>coffee</li>
        <li></li>
        <li></li>
      </ul>

    # Unordered HTML List - Choose List Item Marker
      自行挑選標註

      The CSS 'list-style-type' property is used to define the style of the list item marker. 
      It can have one of the following values:
      Value 	Description
      'disc' 	Sets the list item marker to a bullet (default)
      'circle' 	Sets the list item marker to a circle
      'square' 	Sets the list item marker to a square
      'none' 	The list items will not be marked

      <ul style="list-style-type:circle;">
        <li>coffee</li>
        <li></li>
        <li></li>
      </ul>

    > Ordered HTML List <ol>
      An 'ordered' list starts with the <ol> tag. 
      Each list item starts with the <li> tag.

      The list items will be marked with numbers by default:
        預設項目以'數字'標註

      <ol>
        <li>coffee</li>
        <li></li>
        <li></li>
      </ol>

    > HTML Description Lists <dl>

        HTML also supports description lists.
      A description list is a list of terms, with a description of each term.

      The <dl> tag defines the description list, 
      the <dt> tag defines the term (name), 
      and the <dd> tag describes each term:

      <dl>
        <dt>coffee</dt>
        <dd> 冰咖</dd>
        <dt>牛奶</dt>
        <dd> 溫奶</dd>
        
      </dl>

    > HTML List Tags
      Tag 	Description
      <ul> 	Defines an unordered list
        <li> 	Defines a list item
      <ol> 	Defines an ordered list
        <li> 	Defines a list item
      <dl> 	Defines a description list
      <dt> 	Defines a term in a description list
      <dd> 	Describes the term in a description list

    > HTML Block and Inline Elements

      Every HTML element has a default display value, depending on what type of element it is.

      # Block-level Elements

         * block-level element always starts on a new line, (block-level element總是從新行開始)
            and the browsers automatically add some space (a margin) before and after the element.
            (瀏覽器會自動在block-level element前後加上空白)

         * A block-level element always takes up the full width available 
              (stretches out to the left and right as far as it can).
            (block-level element block-level element總是會延伸到最寬)

          * Two commonly used block elements are: <p> and <div>.
            (兩個最常用的 block-level element 是 <p> & <div>)

            The <p> element defines a paragraph in an HTML document.
              <p> 定義段落

            The <div> element defines a division or a section in an HTML document.
              <div> 定義部分區塊

      # Inline Elements

        An inline element does not start on a new line. (Inline Elements不會從新行開始)

        An inline element only takes up as much width as necessary. (Inline Elements只會佔用必要的寬度)

        This is a <span> element inside a paragraph.
        <span> hello world </span>

        Note: An inline element cannot contain a block-level element!
        注意1: inline element內不能包含 block-level element(區塊元素)
        注意2: 實際上可以( An inline element can contain a block-level element), 
              只是會折行.

      # The <div> Element => block-level elements

        The <div> element is often used as a container for other HTML elements.
        div 常作為其他 HTML elements的容器

        The <div> element has no required attributes, but style, class and id are common.
        div 經常會加上 style, class and id 屬性.

        When used together with CSS, the <div> element can be used to style blocks of content:
          <div style="background-color:black;color:white;padding:20px;">
            <h2>London</h2>
            <p>London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.</p>
          </div> 

      # The <span> Element

          The <span> element is an inline container used to mark up a part of a text, 
            or a part of a document.

          The <span> element has no required attributes, but style, class and id are common.

          When used together with CSS, the <span> element can be used to style parts of the text:

          <p>My mother has 
            <span style="color:blue;font-weight:bold;">blue</span> 
            eyes and my father has 
            <span style="color:darkolivegreen;font-weight:bold;">dark green</span> 
            eyes.
          </p>

      # Chapter Summary

        * A block-level element always starts on a new line and takes up the full width available
        * An inline element does not start on a new line and it only takes up as much width as necessary
        * The <div> element is a block-level and is often used as a container for other HTML elements
        * The <span> element is an inline container used to mark up a part of a text, or a part of a document

      # Using The id Attribute

        * The syntax for id is: write a hash character (#), followed by an id name. 
          Then, define the CSS properties within curly braces {}.

        * In the following example we have an <h1> element that points to the id name "myHeader". 
          This <h1> element will be styled according to the #myHeader style definition in the head section:

        #<CodeSample>

            #myHeader{
              color: black;
              padding: 40px;
              text-align: center;
            }

          </style>

          <h1 id="myHeader"> My Header </h1>

          </CodeSample>

      # Difference Between Class and ID
        A class name can be used by multiple HTML elements, 
          以 . class name 方式命名的 CSS 可以被多個 HTML elements 共用;

        while an id name must only be used by one HTML element within the page:
          以 # id name 方式命名的 CSS 只供單一 HTML elements 專用;

      #<CodeSample>
          <style>
          /* Style the element with the id "myHeader" */
          #myHeader {
            background-color: lightblue;
            color: black;
            padding: 40px;
            text-align: center;
          }

          /* Style all elements with the class name "city" */
          .city {
            background-color: tomato;
            color: white;
            padding: 10px;
          }
          </style>

          <!-- An element with a unique id -->
          <h1 id="myHeader">My Cities</h1>

          <!-- Multiple elements with same class -->

          <h2 class="city">Paris</h2>

        </CodeSample>

    > HTML Iframes
        An HTML iframe is used to display a web page within a web page.

        Tip: It is a good practice to always include a 'title' attribute for the <iframe>. 
              This is used by screen readers to read out what the content of the iframe is.
        註: iframe 最好要加上 title 屬性.

        # HTML Iframe Syntax

          <iframe src="url" title="description"> </iframe>

        # Iframe - Set Height and Width

          * Use the 'height' and 'width' attributes to specify the size of the iframe.
          * The height and width are specified in pixels by default:

            <iframe src="url" title="description" height="200" width="300" > </iframe>
            <iframe src="url" title="description" style="height:200px; width: 300px;" > </iframe>

        # Iframe - Remove the Border

          * By default, an iframe has a border around it.
            預設, iframe 會被外框包住

          * To remove the 'border', add the style attribute and use the CSS border property:

          <iframe src="url" title="description" style="height:200px; width: 300px; border:none;" > </iframe>

          <iframe src="url" title="description" style="height:200px; width: 300px; border: 2px solid red;" > </iframe>

        # Iframe - Target for a Link (a target 連結網頁由 iframe 來顯示, )

          * An iframe can be used as the 'target' frame for a link.

          * The 'target' attribute of the link must refer to the 'name' attribute of the iframe:          

          <iframe src="demo_iframe.htm" name="iframe_a" title="Iframe Example"></iframe>

          <p><a href="https://www.w3schools.com" target="iframe_a">W3Schools.com</a></p> 

          註1: 兩點要注意, <a> 的 'href' 是要顯示網頁路徑, 'target' 則是指向 iframe的 'name' 屬性
          註2: <iframe name ="if" title="要有標題" ></iframe>
               <a href="https://www.nsysu.edu.tw" target="if">中山官網</a>

        # Chapter Summary

          The HTML <iframe> tag specifies an inline frame
          The src attribute defines the URL of the page to embed
          Always include a title attribute (for screen readers)
          The height and width attributes specify the size of the iframe
          Use border:none; to remove the border around the iframe
