    第一章 政府採購法的意義及法律性質
      壹、政府採購法的意義
        $1 (立法宗旨): 為建立政府採購制度, 依公平、公開之採購程序, 提升採購效率與功能, 確保採購品質, 爰制定本法.

        # 時空背景, 加入WTO時, 為簽署WTO的政府採購協定(GPA)前, 須完成政府採購立法, 始符合國際標準.
        # 政府辦理採購時應適用的基本法.
        # 採購法設計主軸即是［程序公開透明］, 從招標前的公開閱覽、招標、決標、簽約到履約.

        # 履約、驗收 => 係［私法事件］, 若有爭議, 依{調解}或{仲裁程序}解決.

          (三)本法所稱選擇性招標, 指以公告方式預先依一定資格條件辦理廠商資格審查後, 再行邀請符合資格之廠商投標.
          (四)本法所稱限制性招標, 指不經公告程序, 邀請2家以上廠商比價或僅邀請1家廠商議價.

          # 機關決標 = 承諾

        二、招標方式
          # 三種: 公開招標(須公告)、選擇性招標(須公告)、限制性招標(不公告, 2家以上比價, 只有1家則議價)

      壹、底價的訂定及訂定原則
        $46 (底價的訂定及訂定時機)
          (一)機關辦理採購, 除本法另有規定外, 應訂定底價.

          (二)前項底價之訂定時機, 依下列規定辦理:
            1. 公開招標應於{開標前}定之.
            2. 選擇性招標應於資格審查後之{下一階段開標前}定之.
            3. 限制性招標應於{議價或比價前}定之.

        一、訂有底價的採購
          # 底價訂定原則
            1. 訂定底價, 應參考工程會的價格資料庫.
            2. 廢標後重行招標, 
                若相關條件有所改變(例:招標文件更改, 市場行情改變), 應重行訂定底價.

            3. 機關可訂定不同的底價, 依採購條件之差異.
            4. 機關不應有操弄底價, 為不當的決標行為.
          
          # 訂定底價程序
            1. 除{重複性採購或未達公告金額的採購}, 得由承辦採購單位逕行簽報核定外,

            2. {限制性招標}之議價, 訂定底價前應先{參考廠商之報價或估價單}.
               此外, 機關仍得{參考其他資料}.

            1. 公開招標: 開標前
            2. 選擇性招標: 資格審查後之下一階段開標前
            3. 限制性招標: 議價前 / 比價開標前
            4. 公開取得廠商書面報價或企劃書: 應於議價或比價前定之.

    第二十四章 終止及解除契約
      壹、終止及解除契約的意義
        # 終止契約: 本於法律規定或契約約定, 行使終止權, 使契約的法律關係向{將來消滅}.
          (注意): 終止契約後, 廠商仍有權受領, 終止時間點前機關應為之給付. 
          (註: 終止時間點後, 就沒有..)

        # 解除契約: 本於法律規定或契約約定, 行使終止權, 使契約的法律關係{溯及於訂約時失效}, {回復訂約前的狀態}.
          (注意): 解除契約後, 涉有債務不履行的一方仍應對他方負起損害賠償責任.
          (註: 解除契約, 就像是沒發生過, 恢復自訂前的狀態.)

      貳、終止及解除契約行使的慎重性
        # 實務上, 有選擇標準:
          1. 契約如就特定情形, 已約定對於契約關係的消滅, 採解除契約或終止契約的方式時, 依契約約定. (註: 如契約有規定, 則從其規定.)
          2. 若契約未約定, 如他方已為的給付, 並無不利的情形而欲繼續保有時, 則{終止契約}的方式, 使{契約關係向後消滅為宜}. 
             (註: 如契約未規定, 但已經開始作了, 而且想保留已完成的, 則採 {終止契約} 方式. )
