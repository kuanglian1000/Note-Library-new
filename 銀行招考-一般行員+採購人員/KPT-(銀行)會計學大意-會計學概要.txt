	第 1 章 - 會計概念與原則
		Unit1- 分工合作的社會
			1. 以物易物的社會, 因下列困難, 不易達到貨暢其流的目標.
				a. 價值認同不易. 例: 雙方價值認同的差異
				b. 交易物的不易分割性. 例: 250公斤米換1頭牛, 只有125公斤時不一定換的到半頭牛.
				c. 有限交易物種. 例: 山區部落不易出現漁產品交換.

			2. 分工合作的社會 => 貨幣制度的產生, 獲得政府官方認可, 成為物品流通的重要媒介, 可達到貨暢其流的目標.
				a. 服務業, 以提供專業服務賺取服務費用及利潤. 例: 銀行, 仲介, 會計師事務所, 顧問業..
				b. 買賣業, 以商品買賣價差賺取利潤. 例: 經銷商, 代理商, 百貨公司, 超級市場..
				c. 製造業, 以製造銷售產品賺取利潤. 例: 煉油廠, 汽車工廠, 家具工廠..
				d. 社會大眾, 透過在上述行業服務賺取薪資; 或自行創立上述公司, 賺取利潤..

		Unit2- 商業組織的型態
			1. 獨資 1個投資人, 稱業主, 獨資企業不具法人資格. 
							法律上, 獨資企業與業主是不可分割的, 應負連帶無限清償責任.
							會計上, 獨資企業與業主是兩個獨立個體, 往來交易要清楚記帳.
			2. 合夥 2個以上投資人, 稱合夥人, 合夥企業不具法人資格. 
							法律上, 合夥企業與合夥人是不可分割的, 應負連帶無限清償責任.
							會計上, 合夥企業與合夥人是兩個獨立個體, 往來交易要清楚記帳.
							合夥企業最好要有契約, 詳述合夥人間的權利義務, 作為日後行為準則.
			3. 公司 公司有 法律個體及合計個體
							型態有: 無限公司(無限清償責任), 有限公司(僅就出資部份負責), 兩合公司(無限+有限, 責任則比照辦理)及股份有限公司
							股份有限公司將企業資本劃分為金額相等的單位(股份), 可公開發行並在市場上流通(稱為股票)

		Unit3- 企業二大能力(獲利能力+償債能力)及三大經濟活動(營運活動+投資活動+財務活動) (財務活動不單指借錢, 還包括還錢..)
			1. 獲利能力是企業透過經濟活動來產生合理收益
			2. 償債能力是指經濟活動衍生的應付債務都能屆期清償完畢
			3. 上述所指的經濟活動又包括以下三種:
				a. 營運活動(現金流入有銷售, 提供勞務; 現金流出有購買原物料, 支付費用)
				b. 投資活動(買賣固定資產, 長期持有上游供應商股票, 更新廠房設備, 處分金融資產..)
				c. 財務活動(企業籌資分配利潤的活動, 負債消長, 股東權益分配, 舉債, 發行新股或公司債..)

		Unit4- 企業的財務結構 (資產 = 負債 + 權益)
			1. 企業為一個經濟個體也是一個會計個體.
			2. 會計上認為企業與業主是分開的個體, 之間的任何交易或往來, 都必須詳實記錄.
			3. 因為企業的資產是業主所投資, 故業主對企業有求償權, 這種求償權稱之為股東權益.
			4. 然而, 企業因賒銷營運所產生的應收帳款是企業對客戶的求償權, 也是企業的資產.
			==
			5. 企業因賒購原物料而積欠供應商的價款, 則是企業的負債.
			6. 這些供應商或融資(借錢)給, 通稱為企業的債權人, 債權人對企業有求償權.
			7. 企業清算時, 剩餘資產償還順序是 (債權人 > 業主或股東) , 因此業主或股東的權益也稱 (剩餘求償權).
			==
			8. 資產增加, 可能是負債(債權人權益)增加而來, 也可能是業主權益增加而來.

		Unit5- 會計恆等式 (資產 = 負債 + 股東權益 + 收入 - 費用) | (資產 + 費用 = 負債 + 股東權益 - 收入)
			1. 收入 > 費用時, 則股東享有 收入 - 費用的差額盈餘, 所以股東權益會增加.
			2. 收入 < 費用時, 則股東承擔 費用 - 收入的差額虧損, 所以股東權益會減少.

		Unit6- 會計的資訊系統
			1. 會計的定義: 「會計是對企業經濟資訊的認定、衡量與溝通的程序, 以協助資訊使用者作出審慎判斷與決策」
			2. (認定): 對經營活動的判斷, 就企業的某項活動研判是否為會計應該記錄與處理的對象.
			3. (衡量): 則是以貨幣為單位評價、記錄該項活動的經濟效果.例: 購買原料的成本? 何時認列?..
								彙集某一期間的經營資訊而對企業的財務狀況與經營成果的評估, 亦屬衡量工作
			4. (溝通): 將企業的財務狀況與經營成果與內外部使用者作有效溝通, 而有效溝通工具就是 ("財務報表") .
			5. 會計資訊的使用者
				a. 內部使用者
					1. 管理階層: 年度整體經營績效
					2. 員工: 公司前景如何
				b. 外部使用者
					1. 股東: 藉由("財務報表"), 了解企業財務狀況, 經營成果, 作為投資參考.
					2. 債權人: 藉由("財務報表"), 評估企業的獲利能力及償債能力, 以降低放款風險.
					3. 政府機關: 藉由("財務報表"), 監督企業財務狀況, 保障投資大眾權益.
					4. 研究機構: 藉由("財務報表"), 掌握趨勢並作出投資建議.

		Unit7- 財務會計的慣例
			財務會計的處理都需遵循下列基本慣例(convention):
			1. 繼續經營慣例
				a. 假設企業在可預見的未來不會解散或清算, 而是以一直經營下去為目標, 且足以持續到履行應盡的義務及預定的計劃完成. <= 上述假設可維持企業營運及財務的穩定性及持久性.
				b. 本慣例影響交易記錄, 如(1)資產成本分期攤提於耐用年限而不一次列為費用; (2)預付費用與預收收益才能列為資產或負債, 分期享用與承擔. <= (費用是放左邊, 跟資產在一起) (收入=收益是放右邊, 跟負債放一起)
				c. 依國際會計準則趨勢, 須揭露固定資產市價漲跌的並在會計上認列損益.

	第2章 - 平時會計處理程序
		Unit2-1 會計循環 (在會計期間內, 依序執行 分錄, 過帳, 試算, 調整, 結帳及編表)
			(1)分錄, 日記帳.
			(2)過帳, 分類帳. 日記帳的分錄彙總至分類帳
			(3)試算, 試算表. 驗證分錄與過帳的正確性. 
			(4)調整, 工作底稿. 會計上某些科目, 是跨越會計期間的, 為使每個科目能正確表示編表時的實際情況, 定期整理或修正, 稱為「調整」
			(5)結帳, 結清損益帳戶(虛帳戶結清歸零), 結轉資產、負責及業主權益帳戶(實帳戶結轉至下一期), 稱為「結帳」
			(6)編表, 會計期間結束後, 將期間內所有交易彙總列表. 
				收入, 費用相關有(損益表); 
				資產、負責及業主權益相關有(資產負債表); 
				現金相關有(現金流量表); 
				業主權益增減有(業主權益變動表)或(保留盈餘表)

		Unit2-2 會計分錄
			交易發生時, 應分析下列各項:
				1. 交易的認定, 是否改變5大要素(資產, 負債, 股東權益, 收入, 費用), 如果不會就不須編分錄.
				2. 影響的科目, 會改變哪些科目
				3. 應作的記錄, 依借貸法則, 寫下借 XX $$ / 貸 XX $$

			分錄種類, 分細分3種:
				1. 一般分錄, 又稱為原始分錄, 日記帳.
				2. 調整分錄, 會計上某些科目, 是跨越會計期間的, 為使每個科目能正確表示編表時的實際情況, 定期整理或修正, 稱為「調整」.
				3. 結帳分錄, 會計期間終了時, 將收入及費用結清轉至「本期損益」帳戶, 以結算會計期間內損益(盈虧); 

		Unit2-3 日記簿

		Unit2-4 會計分錄實例

		Unit2-5 總分類帳與明細分類帳

		Unit2-6 過帳(日記帳的分錄彙總至分類帳)
			分類帳的欄位有: 
			1. 帳戶
			2. 日期
			3. 摘要(常省略)
			4. 借(貸)金額
			5. 日頁(日記簿的頁數, 而日記簿上的類頁則會記載分類帳的頁數, 顯示已過帳)
			6. 滿頁註記

		Unit2-7 試算與試算表
			試算表通常會包括下列各項:
				1. 表頭列
				2. 日期列
				3. 會計科目欄, 借方餘額欄, 貸方餘額欄
				4. 合計列

			試算表的編製, 是以總分類帳上每個帳戶餘額為基礎
				工作分述如下:
				1. 總分類帳各帳戶的整理
				2. 編製試算表
		
		Unit2-8 試算表錯誤追查方法
			試算表錯誤追查方法(若試算表借貸總額不等, 就是試算表有錯誤了..)
				1. 速查法
					a. 若借貸總額差數 / 2 整除, 可能是「重複過帳」或「借貸錯置」
					b. 若借貸總額差數 / 9 整除, 可能是「金額數字移位(例: 15,000 抄為 1,500)」或「金額數字錯置(例: 1,500 抄為 5,100)」
				2. 順查法(分錄, 過帳, 試算表的順向追查)
				3. 逆查法(試算表, 過帳, 分錄的逆向追查)

			查核重點
				1. 試算表查核: 借貸餘額重新加總; 與總分類帳各帳戶核對
				2. 總分類帳查核: 借貸餘額重新加總; 過帳工作是否正確
				3. 日記簿查核

		Unit4-1 買賣業的會計處理
			1. 永續盤存制
				進貨時, 
					借: (存貨) / 貸: (應付帳款)|(現金)

				銷貨時, 會有兩筆分錄: (註: 隨時可取得銷貨成本. 實際盤點時, 若與帳面不符, 會以「存貨盤盈(虧)」調整之)
					(1) 借: (現金)|(應收帳款) / 貸: (銷貨收入)
					(2) 借: (銷貨成本) / 貸: (存貨)

			2. 定期盤存制
				進貨時, 
					借: (進貨) / 貸: (應付帳款)|(現金)

				銷貨時, (註: 實際盤點時, 才會計算銷貨成本)
					借: (現金)|(應收帳款) / 貸: (銷貨收入)

			##<常考/陌生/記憶>#補充教材#
			
			轉帳分錄 => 只影響現金以外的增減 & 不涉及現金收付之分錄
				是指交易的發生影響現金以外之財產、債權、或債務的增減，而(不涉及現金收付之分錄) 
			
			混合分錄 => 不只影響現金以外的增減 & 也涉及現金收付之分錄
				是指交易的發生(不僅涉及現金收付)，同時也影響現金以外之財產、債權、債務的增減變化之分錄
			
			多項式分錄 => 影響二個或二個以上科目之分錄
				是指借方或貸方之任何一方、或借貸雙方，皆擁有二個或二個以上的會計科目之分錄。

			簡單分錄 => 只有一個借方會計項目和一個貸方會計項目之分錄

			一、借項通知單(debit memorandum)：
				1、銀行借項通知單：
					由銀行開立，通知公司：
					((銀行借項通知單通常是在銀行向公司收取手續費、利息費用時開立))。
					
					銀行端(收益增加) => 借: 支票存款(銀行的負債) / 貸: 現金(銀行收益)
					公司端(費用增加) => 借: 手續費用 / 貸: 支票存款
				
				2、一般借項通知單 (進貨退出或折讓證明單)
					#買方要求進貨退回, 所以應付帳款會減少, 故放在借項/借方
					由買方開立，通知其供應商：
					((一般借項通知單通常是賣方的貨品有瑕庛，買方要求減少應付款項時開立))。

					買方 => 借: 應付帳款 / 貸: 賣方貨物
					賣方 => 借: 貨物 / 貸: 應收帳款

					我國採用的"進貨退出或折讓證明單"，其功用與上述的借項通知單相當。

			二、貸項通知單(credit memorandum)：
				1、銀行貸項通知單
					由銀行開立，通知公司：
					銀行貸項通知單通常是在銀行代客戶收到 託收票據 或其他款項時開立。

					銀行端(負債增加) => 借: 託收票據/ 貸: 支票存款(負債)
					客戶端(資產增加) => 借: 支票存款(資產) / 貸: 託收票據

					對公司而言其"支票存款(資產)"已增加(應借記)，並貸記託收票據 或認列收入。
				
				2、一般貸項通知單 (銷貨退回或折讓證明單)
					#賣方同意銷貨退回, 所以應收帳款會減少, 故放在貸項/貸方#
					由賣方開立，通知其買方：
					一般貸項通知單通常是賣方的貨品有瑕庛，賣方同意減少應收款項時開立。
				
					賣方 => 借: 商品增加 / 貸: 應收帳款減少
					買方 => 借: 應付帳款減少 / 貸: 商品減少

					我國採用的"銷貨退回或折讓證明單"，其功用與上述的貸項通知單相當。

			②倒置 
			③貸方$450 誤記入借方 
			④借方$45 誤記入貸方 (O) (正解)

			#網路答案解析, 好神哦#
			
				目標：借方大於貸方900，即借餘900
				思考：借貸必相等，有借必有貸
				(A)移位：多零或少零，例如1,000記錄成100或10,000；舉例來說，借方錯寫為1,000，貸方正確為100，即發生借餘900
				(B)倒置：又稱換位，即數字順序寫錯，例如123記錄成321；舉例來說，借方錯寫為2,100，貸方正確為1,200，發生借餘900
				(C)貸方450寫去借方：原本借方就有450，加上放錯邊的450，借餘900
				(D)借方450寫去貸方：原本貸方就有450，加上放錯邊的450，貸餘900

	CH1-基本概念
  * 會計學有二個假設：
      (1)應計基礎假設-以應計基礎認列收益及費損、
      (2)繼續經營假設-視企業永續經營, 並依假設評估資產及負債之價值
  
  * 會計方程式
      資產 = 負債 + 權益
  
  * 交易會不斷地發生，但不影響會計方程式的平衡
  
  * 主要會計報表有四張：
      1. 綜合損益表(動) - 反應企業報導[期間]經營績效的財務報表，依下列公式編製: 
          收益 - 費損 = 本期淨利(淨損)
      2. 權益變動表(靜) - 權益組成項目變動之財務報表，依下列公式編製: 
          期初權益 + 增資 + 本期淨利(-淨損) - 提取 = 期末權益
      3. 資產負債表(靜) - 反應企業特定[日]財務狀況之財務報表，依下列公式編製: 
          資產 = 負債 + 權益

        註：1,2,3 三張報表關係, 綜合損益表計算本期損益 =(轉入)=> 權益變動表 =(得出)=> 期末權益 =(轉入)=> 資產負債表
        
      4. 現金流量表(動)

  * 會計科目可大略分為五類：
      (1)資產、(2)負債、(3)權益、(4)收益、(5)費損

  * 借貸法則
      (左)借方|(右)貸方
         資產+|資產-
         負債-|負債+
         權益-|權益+
         收益-|收益+ (註:收益增加, 權益增加, 所以借貸放同一邊)
         費損+|費損- (註:費損增加, 權益減少, 所以借貸放另一邊)
